DHTML Tabset Component.

  1. Place the following lines in the <HEAD> of your HTML Document, changing the virtual path names to the files to match those of your server, as needed: 
            
        <LINK REL="stylesheet" HREF="tabs.css">
        <script language="JavaScript" SRC="tabs.js"></script>
        <script language="JavaScript" SRC="tabs_definition.js"></script>
        <script language="javascript"> DrawTabs(); </script>
  2. Edit the "tabs_definition.js" file to set your color preferences, number of tabs, etc.

  3. Edit the "tabs.css" to further define colors, widths, etc.

  4. Create correctly identified <DIV> tags within your HTML document to correspond to the tabs you defined in step 2. Instuctions and examples for these can be found in the tabs_definition.js file, as well as in the example.html file included within this distribution. Any HTML that you like can be included in these tags.

  5. Place this tag at the BOTTOM of your HTML document, after the <BODY> tag 
             <script language="JavaScript" SRC="tabs_bottom.js"></script>

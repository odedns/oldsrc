Available Server Groups.

This is server groups screen. It displays all server groups in the network. A server group is a logical group of two or more servers that can share the same configuration and applications. This makes it easier to distribute configuration and application updates through the network. The user can press the link and edit the server group. The user can also delete and create new server groups.

Utilities For Download

 
Dos Utilities

  • df.zip A UNIX like 'df' utility.
  • ecd.zip An enhanced change dir utility, changes disk drives as well.
  • fdump.zip A small file dump utility I wrote.
  • getch.zip A small utility which returns the keycode as an errorlevel useful for batch files.
  • which.zip A UNIX like which utility I wrote.
  • complete.zip A UNIX tcsh like command line completion utility.

Win32 Utilities

  • ntdf.zip A UNIX like df utility for Win32.
  • wfile.zip Displays file version for Win32 executables.
  • wls.zip A Unix like 'ls' utility for Win32.
  • hlookup.zip A host lookup utility.
